# Agente-IA-ServiceDesk

# 🤖 Agente de Triagem Inteligente - Service Desk (Gemini 2.5)

Este projeto utiliza **Inteligência Artificial Generativa** para automatizar a triagem de tickets de suporte técnico, aplicando conceitos de **ITIL v4** e **Engenharia de Prompt**.

---

## 🚀 Sobre o Projeto
Desenvolvido como parte dos meus estudos em **AI Engineering (FIAP)**, este agente não apenas responde dúvidas, mas atua como um **Analista de Triagem Sênior**. Ele é capaz de interpretar problemas complexos de TI, definir criticidade baseada no impacto ao negócio e sugerir procedimentos técnicos imediatos.

### 🛠️ Principais Funcionalidades:
* **Classificação Automática:** Identifica se o problema é Hardware, Software, Redes, ERP ou Acessos.
* **Matriz de Prioridade:** Define se o ticket é Baixa, Média, Alta ou Crítica (P1), justificando a escolha.
* **Plano de Ação N1:** Gera um passo a passo técnico (SOP) para que o analista de primeiro nível resolva o problema rapidamente.
* **Interface Web Personalizada:** Desenvolvida com Streamlit e estilizada com CSS customizado para uma melhor experiência de uso (UX).

---
